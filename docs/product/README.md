## Product Summary

Cooper is a private AIRES workspace for running voice-assisted meetings, adding project context, generating executive artifacts, supervising background work, and experimenting with operator/computer-use workflows. The core idea is simple: the human talks, Cooper listens, Cooper speaks only when invoked, and the conversation can become documents, diagrams, prototypes, plans, and supervised tasks.
